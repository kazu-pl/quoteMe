# QuoteMe

QuoteMe to nowatorski projekt aplikacji służącej dodawaniu cytatów oraz generowaniu losowo wybranych cytatów, które można następnie zapisać jako PDF.

### Technologie:

- React
- Next
- TypeScript
- Redux Toolkit
- React Hook Form
- Ant Design
- JWT

### Inne:

- Rozwijane na: Node 14.21.3
